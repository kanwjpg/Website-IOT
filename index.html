Cara pakai (Siswa)
1. Pastikan program di MakeCode micro:bit menggunakan blok "Bluetooth UART service".

2. Kirim data sensor dari micro:bit dengan format "NamaSensor:Nilai" (contoh: Sensor cahaya:1023). Dashboard akan otomatis membuat grafik!

3. Untuk menerima perintah dari web, pisahkan teks berdasarkan koma (,) di MakeCode.
